**- RO 필터 제품 WEBSITE Devon Framework**  

web.xml -> Filter 설정을 통해 LocaleFilter.java 호출 
LocaleFilter.java -> redirect301 호출을 통해 /en/main으로 Redirect URL 설정 
global.xml -> DevonFramework의 전역설정으로 interceptor설정이 되어 있음 restStack interceptor를 호출 
devon-framework.xml -> restStack Interceptor에 대한 정의가 되어 있음 Interceptor인 Menu쪽을 호출 
LMenuInterceptor -> HeaderInfo정보 데이터를 담음 
en.xml -> /main에 대한 action이 정의 되어 있음RetrieveMainCmd호출 및 mainEn.jsp 페이지로 Return
